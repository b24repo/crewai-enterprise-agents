# CrewAI Enterprise Agents

Enterprise-grade multi-agent workflows using **CrewAI** + **LangChain**.

## Crews
- **HR Crew** — Resume screening, candidate ranking, onboarding automation
- **Sales Crew** — Lead scoring, email drafting, CRM sync (HubSpot)
- **Support Crew** — Ticket classification, auto-resolution, escalation
- **Finance Crew** — Invoice processing, reconciliation, anomaly detection

## Stack
CrewAI 0.80 · LangChain · OpenAI GPT-4o · Supabase · HubSpot API · Slack

## Quick Start
```bash
cd crewai-enterprise-agents
pip install -r requirements.txt
crewai run
```
